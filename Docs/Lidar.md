# MS200k LiDAR

The MS200k is a low cost but effective lidar and is mounted on the top of the kart

**Specificatons:**
- 12m range
- 4500hz sample rate
- Minimum anglular resolution of 0.4 degrees
- Distance accuracy ±20mm at 12 meters
- Precision <15mm at 12 meters

**ROS2**
The `MS200_ros` node is responsible for operating the lidar. More in-depth documentation for the node can be found within the folder for the node (src/MS200_ros)
